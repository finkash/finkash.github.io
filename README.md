# Personal Portfolio Website – Zorica Nikov

This repository contains the source code for a personal portfolio website built using **HTML and CSS**, showcasing professional interests, technical skills, and photography work.

---

## Overview

The portfolio presents:

* Personal introduction and background
* Technical skills and competencies
* Photography gallery with interactive elements
* Contact form and social media links

The design emphasizes **simplicity, readability, and aesthetic balance**, combining soft color palettes with structured layouts.

---

## Tech Stack

* **HTML5** – page structure and content
* **CSS3** – layout, styling, animations

---

## Project Structure

```
/
│── index.html        # Home page (introduction & overview)
│── skills.html       # Skills and progress visualization
│── photo.html        # Photography gallery
│── contact.html      # Contact form
│
├── style/
│   └── style.css     # Main stylesheet
│
├── img_lq/           # Images used across the site
```

## Key Features

### 1. Responsive Layout

* Centered container design
* Flexible sections using CSS Flexbox

### 2. Navigation Header

* Clean menu with links to all sections
* Hover effects for better UX 

### 3. Hero Banner

* Full-width image with overlay text
* Subtle hover interaction and opacity effects

### 5. Skills Visualization

* Interactive progress bars
* Hover animations for engagement

### 6. Photography Gallery

* Masonry-style layout using CSS columns
* Hover zoom effects and focus transitions
* Modal preview for images (lightbox-style experience)

### 7. Contact Form

* Styled input fields and textarea
* Clean UI for user interaction

### 8. Footer

* Motivational quote
* Social media links (LinkedIn, Instagram, X, 500px)
